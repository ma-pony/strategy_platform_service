# Product Overview

strategy_platform_service 是一个面向数字货币量化交易入门用户的**策略科普展示平台**后端服务，专注于加密货币（BTC、ETH 等主流币种）领域，基于 FastAPI 构建纯后端 RESTful API。

## Core Capabilities

- **经典策略展示**：预设经典量化策略（如均线交叉、RSI、布林带等），用户只读浏览，策略仅通过后台管理维护
- **自动回测与结果展示**：系统定时自动执行 freqtrade 回测，展示收益率、年化收益率、夏普比率、最大回撤、交易次数、胜率等核心指标
- **实时信号展示**：后台定期生成交易信号（Buy/Sell/Hold），缓存至数据库，API 直接读取展示
- **字段级权限控制**：匿名用户可浏览基础信息，Free 用户看到中级指标，VIP 用户解锁全部高级指标（夏普比率、胜率、可信度评分等）
- **AI 市场研报**：展示 AI 生成的数字货币市场分析研报，支持匿名访问
- **后台管理**：通过 sqladmin 管理策略配置、用户数据、研报内容和会员状态

## Target Use Cases

- 数字货币量化交易入门用户浏览学习经典策略的原理与历史表现
- VIP 会员解锁高级指标（胜率、夏普比率、可信度评分）辅助交易决策
- 运营人员通过 sqladmin 后台维护策略、管理用户和发布研报

## Value Proposition

以 freqtrade 作为量化引擎底层，为数字货币交易新手提供经典策略的科普展示和实时信号参考，通过分级会员体系实现商业化运营。平台不提供策略开发或自定义回测功能，专注于"看得懂、学得会"的科普体验。
